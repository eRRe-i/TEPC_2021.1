21.4 μs

Este relatório tem por objetivo apresentar os resultados da paralelização de quatro algoritmos. Estes algoritmos foram disponibilizados pelo professor para que os estudantes realizassem a paralelização utilizando a API Open multi-Processing - OpenMP. Os quatro algoritmos são:

  • Multiplicação de matrizes

  • Decomposição LU

  • Integral

  • MDF (o que quer que isso signifique)
